arrays:
- fixed data type, you need to define the size
- everything is defined by the index values
- you can fetch any index at a constant time

1. access
- o(1)

2. insertion
- best case: o(1)
- worst case: o(n), if you need to shift things

3. delete
- best case: o(1)
- worst case: o(n)

4. traverse
- o(n)

5. searching
- best case: o(n), for a sorted array w linear search
o(log n), for a sorted array w binary search

multidimentional arrays:
- 2d, 3d, 4d, etc.
